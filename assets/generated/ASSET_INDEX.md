# 词域探险生成素材索引

素材由 imagegen 生成，统一为 Q 版 2D 手绘风格。透明版位于本目录，原始品红背景版位于 `raw/`。

运行时压缩素材位于 `assets/runtime/`。游戏实际优先加载 `assets/runtime`，其中背景图位于 `assets/runtime/backgrounds/`，避免运行时读取大图导致卡顿。
